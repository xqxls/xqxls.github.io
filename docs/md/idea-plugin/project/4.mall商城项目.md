## 简介

这是一个mall商城项目(github地址：https://github.com/xqxls/shopping-mall)



## 项目内容

#### 1.项目框架搭建

- SpringBoot环境搭建
- 集成Swagger、Redis、Spring Security
- 集成mbg代码自动生成
- 统一结果封装
- 全局异常处理

#### 2.模块划分

- 前台商城系统
- 后台管理系统

#### 3.前台商城系统功能开发

- 首页内容管理
- 会员登录注册管理
- 会员品牌管理
- 会员商品收藏管理
- 会员浏览记录管理
- 会员优惠券管理
- 会员收货地址管理
- 前台商品查询
- 购物车管理
- 前台订单管理

#### 4.后台商城系统功能开发

- 商品管理
- 订单管理
- 营销管理
- 权限管理
